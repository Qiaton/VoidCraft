# VoidCraft

VoidCraft 是一个围绕虚空能、相位手表、模块技能和相位世界展开的 NeoForge 模组。

当前版本面向 Minecraft `1.21.11` 和 NeoForge `21.11.42`。

## 使用说明

- 中文手册：[docs/zh_cn.md](docs/zh_cn.md)

手册包含入门路线、虚空能来源、机器连接、相位手表、模块、相位浅滩和常见问题。

## 开发运行

```bash
./gradlew runClient
```

如果依赖缓存异常，可以执行：

```bash
./gradlew --refresh-dependencies
```
