# Security Policy

## Supported Versions

Security fixes are handled for the current public development line of VoidCraft. At the moment, that line targets:

| Version | Minecraft | NeoForge | Supported |
| --- | --- | --- | --- |
| 1.0.4 | 1.21.11 | 21.11.42 | Yes |

Older builds may receive fixes when practical, but they are not guaranteed.

## Reporting a Vulnerability

Please do not publish exploit details in a public issue before the vulnerability has been reviewed.

Preferred reporting path:

1. Use GitHub Security Advisories or private vulnerability reporting for this repository if available.
2. If private reporting is not available, open a minimal public issue that says you need to report a security problem, without exploit details or reproduction steps.

Include the affected VoidCraft version, Minecraft version, NeoForge version, and a short summary of the impact. After the report is received, a maintainer will coordinate reproduction details and the fix path.
